# FastByte - Landing Page

Una landing page profesional y moderna para FastByte, empresa especializada en optimización de computadoras e informática.

## 📋 Contenido

La landing page incluye las siguientes secciones:

- **Navegación**: Barra de navegación pegajosa con logo y menú responsivo
- **Hero Section**: Imagen destacada con llamado a la acción principal
- **Servicios**: 4 servicios principales con iconos y descripciones
- **Ventajas**: Razones por las que elegir FastByte
- **Estadísticas**: Números destacados sobre la empresa
- **Contacto**: Formulario de contacto e información de contacto
- **Footer**: Enlaces útiles y derechos de autor

## 🎨 Características

✅ **Responsive**: Diseño adaptable para móviles, tablets y escritorio  
✅ **Moderno**: Colores atractivos y tipografía profesional  
✅ **Interactivo**: Menú móvil, animaciones y efectos hover  
✅ **SEO Friendly**: Estructura HTML semántica y metaetiquetas  
✅ **Optimizado**: CSS y JavaScript ligeros  
✅ **Accesible**: Buenas prácticas de accesibilidad web  

## 📁 Archivos

- `index.html` - Estructura HTML principal
- `style.css` - Estilos CSS responsivos
- `script.js` - Funcionalidades JavaScript
